## 基于 STM32F407VET6 的智能多任务网络天气时钟 (裸机完结版)

## 项目简介
本项目是一款独立开发的智能桌面天气时钟终端。
基于 STM32 纯裸机架构，攻克了传统 delay 阻塞死等的痛点，通过硬件定时器实现了轻量级协同式多任务轮询。整机集成了高精度本地走时、环境传感、Wi-Fi 联网以及动态 UI 渲染功能。

## 硬件外设矩阵
* **核心MCU**：STM32F407VET6 (ARM Cortex-M4，主频 168MHz)
* **显示外设**：ST7789 液晶屏 (240x320 分辨率，硬件 SPI 驱动)
* **网络模块**：ESP32C3   (串口通信，走标准 ESP-AT 指令集)
* **环境传感**：AHT20 温湿度传感器 (I2C 总线，毫秒级时序响应)

## 软件技术架构
1. **时间片轮询状态机**：利用 SysTick 产生 1ms 硬件中断更新 64 位核心时间戳，主循环通过非阻塞状态机并行分发秒级刷时、3秒读取传感器、天级网络对时等高并发任务。
2. **流式 SPI 批处理**：优化 LCD 显示的片选（CS）翻转策略，通过单次拉低、流式数据批处理传输，彻底压榨 SPI2 总线带宽。
3. **动态局部刷新与防御**：自研轻量级字库渲染引擎（支持中英文编码分流、自适应物理步进）；通过动态数据掩码覆盖，实现“数据变动才重绘”，0 显存开销下解决全屏闪烁地狱。
